# Web Candela

Sitio responsive de tres secciones vacías con una navegación flotante que indica dinámicamente la sección visible.

## Desarrollo

```bash
npm install
npm run dev
```

## Verificación

```bash
npm test
npm run test:e2e
npm run build
```
